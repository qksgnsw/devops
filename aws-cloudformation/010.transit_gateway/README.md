# 전송 게이트웨이(Transit Gateway) 개요

---

- AWS VPC 또는 온프레미스 등의 네트워크 자원들과 연결할 수 있는 AWS 라우팅 서비스이다.
- TG를 통해 연결된 네워크는 다른 네트워크와 연결할 필요 없이 AWS TG로만 연결하여 상호 통신할 수 있게 한다.
- 자원 관리가 쉽고, 비용이 절감된다.
